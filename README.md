# GAME Martyn-Engreitz Variant-EFFECTS Evaluator

[Martyn *et al.* (2025)](https://www.cell.com/cell/abstract/S0092-8674(25)00352-6) dataset measured the effects of 783 variants in Jurkat T cells and THP-1 monocytes cells. This Evaluator is used to evaluate models' abilities to predict variant effects. We extract 2000bp around the indel and request log scale expression predictions for the reference and mutated sequence.
